# 🎓 Student Performance Prediction

An End-to-End Machine Learning project that predicts student exam scores based on academic, personal, and environmental factors using **Python, Scikit-Learn, and Streamlit**.

---

## 📌 Project Overview

This project analyzes various student-related factors such as study hours, attendance, motivation level, sleep hours, parental involvement, and more to predict the expected exam score.

The project demonstrates a complete Machine Learning workflow from data preprocessing to model deployment using Streamlit.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/jiveshai-07/student-performance-prediction.git
```

Move into the project folder:

```bash
cd student-performance-prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
